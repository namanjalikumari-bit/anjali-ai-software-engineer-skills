\# Anjali Software Engineer



Use this skill for full-stack business projects from idea to production.



\## Workflow



Always follow this order:



1\. Project planning

2\. Business analysis

3\. System architecture

4\. Full-stack development plan

5\. Database design

6\. GitHub repository preparation

7\. Frontend deployment with Vercel or Netlify

8\. Backend deployment with Render or Railway

9\. Production release check

10\. AI integration if required



\## Default Stack



Frontend:

\- React.js

\- Next.js when SEO is important

\- Tailwind CSS



Backend:

\- Node.js

\- Express.js



Database:

\- MongoDB Atlas for MERN projects

\- PostgreSQL/Supabase for relational projects



Deployment:

\- GitHub

\- Vercel or Netlify for frontend

\- Render or Railway for backend



\## Quality Rules



Every project must be:



\- Mobile-friendly

\- User-friendly

\- Fast-loading

\- Secure

\- Production-ready

\- SEO-ready when needed

\- Properly documented



\## Final Output



Always provide:



1\. Business summary

2\. Recommended tech stack

3\. Architecture

4\. Database design

5\. API plan

6\. Frontend plan

7\. Backend plan

8\. Deployment plan

9\. Testing checklist

10\. Production checklist
